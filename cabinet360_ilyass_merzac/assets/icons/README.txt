Cabinet360 PWA Icons
====================

To generate the PNG icons:

Option 1: Use the HTML Generator (Easiest)
-------------------------------------------
1. Open the file: create_icons.html (in the root folder)
2. Click the "Download Icons" button
3. Move the downloaded icon-192x192.png and icon-512x512.png to this folder (assets/icons/)
4. Delete create_icons.html after use

Option 2: Use the Python Script
--------------------------------
1. Install Pillow: pip install Pillow
2. Run: python generate_icons.py
3. Icons will be automatically created in this folder

Option 3: Create Your Own Icons
--------------------------------
Create two PNG files:
- icon-192x192.png (192x192 pixels)
- icon-512x512.png (512x512 pixels)

Recommended design:
- Background: Dark (#1a1a1a)
- Primary color: Gold (#D4AF37)
- Theme: Legal/Law firm (scales of justice)
- Include "CABINET360" text

Temporary SVG files (icon-192x192.svg and icon-512x512.svg) are provided as references.
